## Overview

USD Search NIM is an NVIDIA NIM microservice announced in January 2026 that enables developers to search through massive libraries of OpenUSD, 3D and image data using natural language or image inputs.

## Capabilities

- Natural language search across 3D assets
- Image-based similarity search
- OpenUSD scene search
- Multi-modal content discovery
- Integration with asset management systems

## Search Modalities

- Text queries ("find all red cars")
- Image queries (visual similarity)
- Combined text+image queries
- Semantic 3D asset search

## Use Cases

- 3D asset library management
- Digital twin asset discovery
- Game development asset search
- Manufacturing part libraries
- Architectural model databases

## Technical Architecture

- Optimized for NVIDIA GPUs
- Multimodal embedding models
- Vector search backend
- Real-time indexing

## Integration

Seamlessly integrates with NVIDIA Omniverse and OpenUSD workflows for industrial metaverse applications.

## Overview

imvectordb is a super simple in-memory vector DB for Node.js. With a compressed (ZIP) file size of just 3KB, it's perfect for quickly building prototypes or small-scale applications.

## Features

- Ultra-lightweight at just 3KB compressed
- Automatic embedding using OpenAI's text-embedding-ada-002 model
- Simple, intuitive API
- Perfect for prototypes and small-scale applications
- In-memory storage for fast access

## Installation & Setup

The library automatically uses OpenAI's "text-embedding-ada-002" model for embedding, so you'll need to set the OPENAI_API_KEY environment variable.

## Use Cases

- Rapid prototyping of vector search features
- Small-scale applications with minimal dependencies
- Learning and experimentation with vector databases
- Lightweight embedding and similarity search

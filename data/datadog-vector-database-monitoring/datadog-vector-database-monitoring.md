## Overview

Datadog integration with Zilliz Cloud provides comprehensive monitoring and observability for vector database infrastructure, allowing organizations to monitor vector databases alongside their entire technology stack.

## Key Metrics

- Queries Per Second (QPS)
- Vectors Per Second (VPS)
- Query latency measurements
- Slow query detection
- Failure rates
- Resource utilization
- Index health

## Features

- Real-time monitoring dashboards
- Alert configuration for anomalies
- Performance trend analysis
- Integration with full observability stack
- Custom metric tracking
- Historical data analysis

## Integration

- Native Zilliz Cloud integration
- Automated metric collection
- Pre-built dashboards
- Custom alerting rules
- API access for programmatic monitoring

## Use Cases

- Production vector database monitoring
- Performance optimization
- Capacity planning
- Issue detection and debugging
- SLA compliance tracking

## Benefits

- Unified observability across infrastructure
- Proactive issue detection
- Performance insights
- Reduced troubleshooting time
- Better operational visibility

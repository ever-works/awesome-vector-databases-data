## Overview

FiftyOne is an open-source toolkit for building high-quality datasets and visual AI models. It provides a computer vision interface for vector search with powerful visualization and curation capabilities.

## Features

- **Natural Language Search**: Query datasets using natural language
- **Vector Database Integrations**: Native support for Qdrant, Pinecone, LanceDB, Milvus, and more
- **Billion-Scale Search**: Query across databases with billions of images
- **Visualization**: Visual interface for search results
- **Multi-Modal**: Support for images, video, and point clouds
- **Dataset Management**: Comprehensive dataset operations
- **Model Integration**: Works with major ML frameworks

## Supported Vector Databases

- Elasticsearch
- LanceDB
- Milvus
- MongoDB Atlas Vector Search
- Pinecone
- Qdrant
- Redis Vector Search

## Use Cases

- Computer vision model development
- Image similarity search
- Video data curation
- Pattern detection in visual data
- Multimodal search applications
- Dataset quality improvement

## Integration

FiftyOne acts as a central hub, orchestrating workflows from loading and curating datasets to querying vector search databases and visualizing matches. Works with Databricks Vector Search and Twelve Labs.

## Performance

Enables users to create indexes with compute_similarity() and search across entire vector databases efficiently.

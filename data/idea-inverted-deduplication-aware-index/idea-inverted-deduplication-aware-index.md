# IDEA: Inverted Deduplication-Aware Index

Research paper presenting IDEA, an inverted deduplication-aware index that compares physical vs. logical indexing approaches for vector search. Published at the 22nd USENIX Conference on File and Storage Technologies (FAST 24) in 2024.

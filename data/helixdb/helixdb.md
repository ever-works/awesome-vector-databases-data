## Overview

Graph-vector database in Rust for RAG and AI apps.

## Features
- Graph + vector search
- High performance Rust implementation
- Open-source (AGPL-3.0)

## Use Cases
- Knowledge graphs
- KG-RAG
- AI/ML workflows with relational data

## Comparisons
Vs pure vector DBs: Built-in graph capabilities for relationship traversal.

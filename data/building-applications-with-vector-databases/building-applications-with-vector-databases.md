## Overview

DeepLearning.AI's short course teaches how to create six exciting applications of vector databases and implement them using Pinecone, covering RAG, recommender systems, and advanced search techniques.

## Applications Covered

1. **RAG for LLM Enhancement**: Augment language models with external knowledge
2. **Recommender Systems**: Combine semantic search with RAG
3. **Hybrid Search**: Find items using both images and descriptive text
4. **Semantic Search**: Implement efficient similarity search
5. **Anomaly Detection**: Identify outliers in vector spaces
6. **Question Answering**: Build intelligent Q&A systems

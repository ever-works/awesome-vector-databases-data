## Overview

Redis module for full-text, secondary indexing, vector similarity (KNN/filtered/range).

## Features

- Vector similarity: KNN, filtered KNN
- Full-text with tf-idf, phonetic, stemming
- Aggregations, geospatial
- Cluster scaling (Enterprise)

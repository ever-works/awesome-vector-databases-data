## Overview

VAST CNode-X is a GPU-accelerated server from VAST Data designed for demanding enterprise AI workloads. It combines the VAST AI OS with NVIDIA data-processing libraries (cuVS, cuDF) and onboard NVIDIA RTX PRO 6000 Blackwell Server Edition GPUs.

## Architecture

- Combines VAST AI OS with NVIDIA data-processing libraries
- Features NVIDIA RTX PRO 6000 Blackwell Server Edition GPUs
- Leverages the NVIDIA AI Data Platform reference design
- Designed for multi-agent systems, video-reasoning, and high-volume RAG workloads

## Capabilities

- Accelerates data vectorization, vector search, and inference
- Sustained ingestion rate of 1 million vectors/second into the vector database
- Benchmarking at high throughput, performance, and recall across 50 billion vectors
- Indexing lifecycle 4.5x faster than CPU-only approaches
- Supports concurrent SQL query execution with GPU-accelerated operators

## Use Cases

- Real-time RAG pipelines
- Recommendation systems
- Video-reasoning over public safety streams
- Finance, cybersecurity, genomics/life sciences, retail, and chemistry applications

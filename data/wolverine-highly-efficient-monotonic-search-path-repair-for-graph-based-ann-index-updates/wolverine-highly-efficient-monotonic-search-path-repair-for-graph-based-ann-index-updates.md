## Overview

Wolverine addresses the challenge of efficiently updating graph-based ANN indexes by repairing monotonic search paths after data modifications.

## Key Contributions

- Efficient incremental update for graph-based ANN indexes
- Monotonic search path repair after insertions and deletions
- Maintains search quality after dynamic updates
- Published in VLDB 2025

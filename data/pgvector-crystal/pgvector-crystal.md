## Overview

pgvector-crystal offers Crystal language bindings for pgvector, enabling vector operations in PostgreSQL from Crystal applications.

## SDK Features

- Integration with crystal-pg PostgreSQL driver
- Vector embeddings support (OpenAI, Cohere binary)
- Hybrid search with RRF (Ollama)
- Sparse vector search
- Similarity queries (cosine, L2, etc.)

## Use Cases

- App integration for Crystal/Phoenix stacks
- RAG and semantic search
- Embeddings storage/querying

## Comparisons

- Community Crystal client vs. official Python/JS/.NET or raw SQL

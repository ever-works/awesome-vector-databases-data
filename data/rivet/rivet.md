## Overview

Rivet is an open-source visual programming environment for building AI agents with LLMs. Developed by Ironclad, it's a desktop application for creating complex AI agents and prompt chaining with visual workflows.

## Key Features

- **Visual Node-Based Interface**: Drag-and-drop canvas for building AI workflows by connecting nodes
- **Real-Time Debugging**: View input/output of every node and AI responses in real-time
- **Open Source**: Available on GitHub with both desktop application and TypeScript libraries
- **Integration**: Can be embedded in applications for bidirectional communication between Rivet and application code
- **Export to Code**: Prototype visually, then export to TypeScript for production

## How It Works

Create complex AI agent chains and prompt chains by connecting nodes visually. The node-based editor enables configuration and debugging of AI workflows without extensive coding.

## Real-World Usage

Ironclad uses Rivet to power their own Contract AI assistant. Adopted by companies including:
- Willow Servicing (mortgage servicing platform)
- Attentive
- Sourcegraph

## Use Cases

- Rapid prototyping of AI agents
- Visual debugging of prompt chains
- Building conversational AI systems
- Creating multi-step reasoning workflows
- Team collaboration on AI development

## Target Audience

Designed for teams looking to rapidly prototype, debug, and collaborate on AI agents without writing extensive code, while maintaining ability to export for production deployment.

## Overview

VectorAI DB is designed to run vector search wherever your application lives - on your laptop, at the edge, in your data center, or in air-gapped environments. It has no cloud dependency and no connectivity assumptions.

## Key Features

- Single-node deployment at remote locations and edge devices
- Sub-15ms query latency for real-time decision making
- Operates independently during network outages with optional synchronization
- Facial recognition for security, access control, and identity verification at the edge
- Processes biometric data locally without cloud dependencies for privacy and compliance
- No network round-trips or cloud API delays required

## Use Cases

- Store checkout hardware with unstable connectivity
- Factory-floor machinery and industrial IoT
- Air-gapped environments requiring local data analytics
- Edge devices requiring autonomous behavior
- Security and access control systems

## Availability

VectorAI DB addresses the native edge deployment gap with a database that accounts for the physical and network realities of offline scenarios.

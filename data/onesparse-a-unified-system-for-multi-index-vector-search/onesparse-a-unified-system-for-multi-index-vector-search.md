# OneSparse: A Unified System for Multi-index Vector Search

Research paper presenting OneSparse, a unified system for multi-index vector search. Published at the Companion Proceedings of the ACM on Web Conference 2024, it addresses the challenge of efficient vector search across multiple index structures.

## Overview
DuckDB enables low-latency vector search via SQL on edge devices.

## Features
- HNSW vector search extension
- In-process embeddable
- Parquet/CSV support
- Multi-platform

## Use Cases
- Desktop edge AI analytics
- IoT data processing
- Local CV inference

## Comparisons
- On-device SQL vs cloud Qdrant
- Lighter than LanceDB for analytics

## Core Technology

### Mem0 Base Algorithm
Scalable memory-centric algorithm that:
1. Dynamically extracts key conversational facts
2. Stores facts in optimized memory structures
3. Retrieves relevant context efficiently
4. Self-improves over time

### Mem0ᵍ (Graph-Enhanced Variant)
Layered architecture combining:
- **Vector Store**: Fast semantic similarity search
- **Graph Store**: Rich multi-session relationships

#### Graph Processing Pipeline
1. **Entity Extraction**: Identifies entities as graph nodes
2. **Relations Generation**: Infers labeled edges between entities
3. **Graph Construction**: Transforms text into structured graph
4. **Hybrid Retrieval**: Leverages both vector and graph queries

## Overview

Dify is an open-source platform for building LLM applications that provides a visual interface for creating AI workflows, RAG pipelines, and autonomous agents. It's designed to help developers quickly move from prototype to production.

## Features

- **Visual Workflow Builder**: Drag-and-drop interface for building AI pipelines
- **RAG Pipeline**: Built-in retrieval-augmented generation capabilities
- **Agent Framework**: Support for autonomous AI agents
- **Model Management**: Unified interface for multiple LLM providers
- **Knowledge Base**: Integrated document management and vector storage
- **Observability**: Built-in monitoring and debugging tools
- **API Generation**: Automatic API creation for workflows
- **Multi-Tenant**: Support for multiple users and organizations

## Integration

Integrates with Langfuse for enhanced observability and supports major vector databases including Qdrant, Weaviate, Pinecone, and Milvus.

## Use Cases

- Rapid prototyping of LLM applications
- Building customer support chatbots
- Creating knowledge base assistants
- Developing AI agents for automation

## Deployment

Can be self-hosted or used through Dify Cloud. Docker deployment available for easy setup.

## Pricing

Open-source and free for self-hosting. Dify Cloud offers managed hosting with usage-based pricing.

## Overview

VectorDBZ is an enterprise-grade desktop application for managing and analyzing vector databases with interactive visualizations.

## Supported Databases

- Qdrant
- Weaviate
- Milvus
- ChromaDB
- Pinecone
- pgvector
- Elasticsearch

## Key Features

### Interactive Visualizations

Powered by ECharts for analyzing vector space relationships through dimensionality reduction, providing insights into how vectors are distributed and clustered.

### Live Monitoring

Monitor analysis progress in real-time, helping understand performance characteristics and identify bottlenecks.

### Management Capabilities

- Manage multiple vector databases from a single interface
- Analyze vector space relationships
- Optimize query performance

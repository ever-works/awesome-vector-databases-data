## Overview

SeRF introduces a segment graph data structure for efficient range-filtering approximate nearest neighbor search, supporting queries that combine vector similarity with attribute range constraints.

## Key Contributions

- Segment graph construction for range-filtering search
- Efficient hybrid query processing
- Combines vector similarity with attribute range filtering
- Published in SIGMOD 2024

## Publication

- **Venue**: SIGMOD 2024
- **Authors**: Zuo et al.
- **Abbreviation**: SeRF

## Features

- Range-filtered ANNS support
- Segment-based graph index
- Efficient attribute constraint handling

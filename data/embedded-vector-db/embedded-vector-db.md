## Overview
embedded-vector-db delivers low-latency hybrid vector/full-text search on-device.

## Features
- HNSW ANN + BM25 hybrid (RRF fusion)
- Multi-namespace CRUD
- Metadata filtering
- Disk persistence

## Use Cases
- IoT real-time search
- Edge RAG apps
- CV inference Node.js

## Comparisons
- On-device vs cloud Qdrant
- Lighter, no-server alternative

## Overview

LlamaHub is an open-source repository containing data loaders that can easily plug and play into any LlamaIndex application, with over 160 data connectors created by the community.

## Features

- 160+ data connectors
- Community-driven contributions
- Easy plug-and-play integration
- Wide format support (PDFs, databases, APIs)
- Comprehensive documentation
- Active development

## Connector Types

### Document Formats
- PDF documents
- Word documents
- Markdown files
- HTML pages
- Text files

### Data Sources
- Notion databases
- Google Drive
- Confluence
- Slack
- GitHub repositories
- SQL databases
- APIs and webhooks

## Integration

- Native LlamaIndex support
- Compatible with LangChain
- Simple pip install
- Minimal configuration required
- Standardized interfaces

- Document processing workflows

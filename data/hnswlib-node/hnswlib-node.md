## Overview

hnswlib-node is a JavaScript/TypeScript SDK providing Node.js bindings to HNSWLib for approximate nearest-neighbor vector search.

## SDK Features

- Async operations for non-blocking queries
- Multi-threading support
- Distance metrics: L2 (Euclidean), Cosine, Inner Product
- File persistence: save/load indices to disk
- Custom filter functions
- LangChain vector store integration

## Use Cases

- Integrate vector search into Node.js web apps
- Serverless RAG pipelines
- Real-time recommendation systems
- Semantic search backends

## Comparisons

- Official community JS bindings vs. Python hnswlib
- Lighter than full DBs (e.g., Chroma, LanceDB) for JS stacks

## Installation

```bash
npm install hnswlib-node
```

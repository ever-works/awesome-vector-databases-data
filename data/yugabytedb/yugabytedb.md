# YugabyteDB

PostgreSQL-compatible distributed SQL database with built-in vector search capabilities using HNSW indexing. Supports ACID transactions, horizontal sharding, and auto-rebalancing to scale vector workloads from millions to billions of embeddings for AI and GenAI applications. Combines vector similarity search with relational queries including joins and aggregations.

## Overview

CrewAI orchestrates role-based AI agents ('crews') that collaborate using shared vector memory and tools to solve complex tasks.

## Vector Integration

Integrates with vector stores for long-term memory and knowledge retrieval:
- Entity memory backed by vector databases
- Context-aware retrieval for agent decision-making
- Compatible with Qdrant, Pinecone via LangChain integrations

## Features

- **Tool Calling**: 100+ built-in tools (search, databases, APIs)
- **Memory**: Short-term, long-term, entity, contextual memory with vector backing
- **Agents**: Hierarchical crews with planning, delegation, and execution

## Use Cases

- Multi-agent chatbots for customer support
- Autonomous research agents
- Workflow automation crews

## Comparisons

CrewAI focuses on multi-agent collaboration vs LangChain's chain-based orchestration.

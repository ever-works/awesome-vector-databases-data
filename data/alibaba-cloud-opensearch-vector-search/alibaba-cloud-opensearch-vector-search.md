# Alibaba Cloud OpenSearch Vector Search

Alibaba Cloud OpenSearch is a fully managed intelligent search platform supporting vector search, semantic understanding, and multimodal retrieval. It integrates with DingTalk AI and uses ANN algorithms for high-performance similarity search.

## Features

- Approximate nearest neighbor (ANN) algorithms for vector search
- Integration with DingTalk AI
- Managed service with millisecond latency
- Support for text, image, audio, and video retrieval

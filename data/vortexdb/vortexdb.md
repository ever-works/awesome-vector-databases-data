## Overview
VortexDB supports flexible low-latency edge vector search.

## Features
- HNSW/KD-Tree/Flat indexers
- Multiple distances
- RocksDB/in-memory
- HTTP/gRPC/TUI

## Use Cases
- IoT real-time similarity
- Edge RAG
- Embedded CV

## Comparisons
- Modular edge vs cloud Qdrant
- Multiple index options

## Pricing
Free open-source (Apache 2.0).

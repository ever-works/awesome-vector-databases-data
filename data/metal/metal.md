## Overview

Metal is a production-ready, fully-managed ML retrieval platform for professional users and developers seeking a tool to find meaning in unstructured data with embeddings. It includes a vector database and critical components for common LLM applications.

## Features

- **Managed Service**: Build AI products without managing infrastructure - hosting and high availability handled automatically
- **REST API**: Simple and straightforward API available at https://api.getmetal.io
- **ANN Queries**: Convenient /search endpoint for running approximate nearest neighbor queries
- **Integrations**: Built-in support for OpenAI, CLIP, and other embedding models
- **SDK Support**: TypeScript/JavaScript SDK available via npm (@getmetal/metal-sdk)

## API Capabilities

- **Text and Image Indexing**: Support for indexing from URLs or Base64-encoded data
- **Search with Filtering**: Advanced search with metadata filtering capabilities
- **Metadata Support**: Custom metadata fields for enriched search results
- **Authentication**: Metal API Keys for secure access to API and SDKs

## Integration

Metal integrates with:
- JavaScript and TypeScript applications
- LangChain framework
- OpenAI embeddings
- Various LLM providers

## Use Cases

- Building LLM applications with memory
- Semantic search over unstructured data
- Recommendation systems
- Document similarity search
- Image search with CLIP embeddings

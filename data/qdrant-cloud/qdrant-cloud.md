## Overview

Qdrant Cloud is the fully managed, hosted deployment of the Qdrant vector database. It provides production-ready infrastructure for vector search and AI workloads without requiring users to manage servers or underlying cloud resources.

## Features

### Deployment & Infrastructure
- **Run anywhere**: Deploy on major cloud providers:
  - AWS
  - Google Cloud
  - Microsoft Azure
- **Global regions**: Choose regions globally for lower latency and proximity to your data and applications.
- **Quick cluster provisioning**: Create and deploy Qdrant clusters in seconds via the Qdrant Cloud Console.
- **Elastic scaling**: Clusters can scale up as workload and data volume grow (details not fully specified, but implied by “scales up as needed”).

### Setup & Management
- **Simple setup**: Streamlined, console-based deployment flow, suitable for rapid prototyping and production use.
- **Managed operations**: Qdrant handles hosting, maintenance, and operational tasks such as upgrades and infrastructure management.

### Performance & Resource Efficiency
- **Efficient resource management**:
  - Built-in compression options to reduce memory usage.
  - Capability to offload data to disk to optimize RAM consumption and cost.

### Reliability & Maintenance
- **Zero-downtime upgrades**:
  - Perform cluster upgrades and scaling operations without interrupting service.
  - Supports continuous operation during model or configuration updates.
- **Continuous backups**:
  - Automated backups of data.
  - Configurable backup policies.
  - Ability to restore data to previous states.

### Ecosystem & Access Options
- **Marketplace availability**:
  - AWS Marketplace
  - Google Cloud Marketplace
  - Microsoft Azure Marketplace
- **Documentation**: Extended technical documentation for advanced features and configuration (referenced as an external resource).
- **Enterprise deployment options**: For organizations needing additional control, Qdrant is also offered as Hybrid Cloud and Private Cloud (full on-premise) solutions (outside core Cloud product but related).

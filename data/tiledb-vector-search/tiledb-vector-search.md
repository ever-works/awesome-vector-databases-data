## Overview

TileDB Vector Search is a vector search solution leveraging the TileDB array storage format for high-performance similarity search.

## Key Components

- **apis/python**: Python bindings and APIs for integration.
- **demos**: Example applications and use cases.
- **documentation**: Comprehensive guides and references.
- **src**: Core C++ implementation.
- **.github/workflows**: CI/CD pipelines.

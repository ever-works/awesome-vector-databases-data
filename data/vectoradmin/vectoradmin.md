## Overview

VectorAdmin is the universal vector database management system, UI, and tool suite designed for developers building applications with vector databases across multiple platforms.

## Supported Databases

- Pinecone
- Chroma
- Qdrant
- Weaviate
- Additional vector databases

## Features

- Universal management interface
- Multi-database support
- Centralized administration
- Data visualization
- Query interface
- Collection management
- Cross-platform compatibility

## Key Capabilities

- Switch between different vector databases
- Unified interface for multiple platforms
- Data import/export
- Query testing and debugging
- Performance monitoring
- Schema management

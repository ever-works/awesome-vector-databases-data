## Overview

VectorDB.js is a simple in-memory vector database for Node.js. It's an easy way to do text similarity.

## Features

- Works 100% locally and in-memory by default
- Uses hnswlib-node for simple vector search
- Uses Embeddings.js for simple text embeddings
- Supports remote embedding services like OpenAI and Mistral
- Can store any valid JavaScript object along with the embedding
- Simple API with configurable dimensions (default 384) and initial size (default 100)

## Use Cases

Perfect for developers who need:
- Text similarity search in Node.js applications
- Local, in-memory vector operations
- Quick prototyping with vector search
- Lightweight vector database without external dependencies

## Installation

Available as `@themaximalist/vectordb.js` on npm.

## Overview

Nuclia is an AI Search and RAG-as-a-Service platform providing semantic search and generative AI capabilities for enterprise applications. Acquired by Progress on July 1, 2025, now operating as Progress Agentic RAG.

## Key Features

- **RAG-as-a-Service**: Complete RAG pipeline management
- **Semantic Search**: AI-powered search over unstructured data
- **NucliaDB**: Open-source vector database core
- **Multimodal Support**: Text, documents, images, and more
- **Generative Answers**: AI-generated responses from your data
- **Enterprise-Ready**: SOC 2 compliant, production-grade

### Cloud Deployment
- **Data Regions**: Europe or USA
- Fully managed by Nuclia/Progress

### Hybrid Deployment
- Processing in Nuclia cloud

### On-Premise

## NucliaDB (Open Source)

- Open-source vector database
- Powers Nuclia's RAG platform
- Available on GitHub: nuclia/nucliadb
- Can be self-hosted independently
- Apache 2.0 or similar permissive license

## Overview

OpenSearch Vector Search provides vector similarity and AI search within the OpenSearch engine, storing/searching embeddings alongside indexed data for semantic/RAG apps.

## Features

- Raw/hybrid/multimodal/neural sparse search
- k-NN/ANN indexes
- Automatic embedding generation via ML Commons
- Ingest pipelines for vectors

## Use Cases

- Semantic search
- RAG pipelines
- Recommendations
- Conversational search

## Pricing

Open-source; hosting costs vary.

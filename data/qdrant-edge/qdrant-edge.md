## Overview

Embedded vector search for edge devices.

## Features

- Filtered ANN (HNSW)
- Payload filtering
- Multi-modal support
- Disk-persistent
- Offline edge deployment
- High QPS low-latency

## Use Cases

- Real-time recommendations
- Semantic search on-device

## Comparisons

- Lighter, Rust perf vs Weaviate
- Open-source alt vs Pinecone

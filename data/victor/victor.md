## Overview
Victor enables low-latency vector ops in web/edge envs.

## Features
- WASM browser support
- Efficient storage (1.5KB/1536d)
- PCA compression
- JS/Rust APIs

## Use Cases
- Browser IoT AI
- Edge semantic search
- Real-time CV web

## Comparisons
- Web/edge vs cloud Qdrant
- Much smaller footprint

## Pricing
Free open-source.

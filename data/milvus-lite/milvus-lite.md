## Overview

Milvus Lite is a pip-installable, embedded vector DB library for local Python environments.

## Features

- Zero extra dependencies (pip install pymilvus)
- Optional persistence
- HNSW and IVF indexes
- Real-time inserts and queries
- LangChain/LlamaIndex integration

## Use Cases

- Prototyping AI apps on laptops/notebooks
- Edge devices and mobile AI

## Comparisons

- Faster startup than Qdrant client libraries
- Simpler setup than full Milvus cluster, scalable path vs Chroma

## Pricing

Free and open-source (Apache 2.0).

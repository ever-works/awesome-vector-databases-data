## Overview

Meilisearch Vector Search integrates vector capabilities into its search engine for hybrid and semantic search in AI applications.

## Features

- **Hybrid Search**: Combines keyword with vector search.
- **Semantic Search**: Natural language understanding.
- **Multi-format**: Text, images, audio, video.
- **Integrations**: OpenAI, Cohere, LangChain, Hugging Face.
- **Filters & Facets**: Advanced filtering.
- **Deployment**: Self-hosted or cloud.

## Use Cases

- Q&A bots
- Product recommendations
- Contextual search

# RTNN: Accelerating Neighbor Search Using Hardware Ray Tracing

Research paper by Yuhao Zhu presenting RTNN, a novel approach that leverages hardware ray tracing capabilities to accelerate approximate nearest neighbor search. Published at the 27th ACM SIGPLAN Symposium on Principles and Practice of Parallel Programming in 2022.

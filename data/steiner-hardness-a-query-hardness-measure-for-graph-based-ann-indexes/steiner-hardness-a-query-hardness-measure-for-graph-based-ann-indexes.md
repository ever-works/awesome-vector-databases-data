## Overview

Introduces a hardness metric for graph-based ANN search queries based on Steiner tree analysis.

## Key Contributions

- Novel query hardness measure for graph-based ANNS
- Characterizes query difficulty through graph topology
- Enables adaptive search strategies based on predicted hardness
- Published in VLDB 2025

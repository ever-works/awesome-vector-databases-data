## Overview

LIRA introduces a learning-based, query-aware partition framework for large-scale approximate nearest neighbor search.

## Key Contributions

- Learned partitioning based on query patterns
- Adapts to query distribution for improved search
- Scalable to large vector datasets
- Published in WWW 2025

## Publication

- **Venue**: WWW 2025
- **Authors**: Zeng et al.
- **Abbreviation**: LIRA

## Features

- Query-aware learning for partition optimization
- Dynamic partitioning strategies
- Improved search efficiency on large-scale datasets

## Overview

ScaNN introduces anisotropic vector quantization combined with asymmetric distance computation for fast large-scale vector similarity search.

## Key Contributions

- Anisotropic vector quantization for high-dimensional search
- Product quantization with optimized codebooks
- Asymmetric distance computation for improved accuracy
- Achieves state-of-the-art speed-accuracy tradeoffs

## Publication

- **Venue**: ICML 2020
- **Authors**: Guo, Kakade, Krishnamurthy, Khanna
- **Abbreviation**: ScaNN

## Impact

ScaNN became a core component of Google's large-scale retrieval systems, enabling fast semantic search and recommendation at scale.

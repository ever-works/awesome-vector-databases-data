## Overview

USD Code NIM is an NVIDIA NIM microservice announced in January 2026 that answers general knowledge OpenUSD questions and automatically generates OpenUSD-Python code based on text prompts.

- Generates OpenUSD Python code from natural language
- Assists with 3D workflow automation
- Integrates with digital twin development pipelines

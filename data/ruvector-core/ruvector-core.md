## Overview
ruvector-core provides low-latency edge vector engine.

## Features
- HNSW + SIMD distances
- Adaptive compression
- Multi-threaded QPS

## Use Cases
- IoT real-time retrieval
- Edge RAG agents
- CV on-device

## Comparisons
- Edge SIMD vs cloud Qdrant
- High QPS on CPU

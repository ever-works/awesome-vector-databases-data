## Overview

VectorDBBench is an open-source tool designed for performance and cost-effectiveness comparison of vector databases, helping users make informed decisions about which database fits their needs.

## Key Features

- Standardized benchmarks across databases
- Visual interface for results
- Reproducible testing
- Cost-effectiveness analysis
- Support for major vector databases

## Supported Databases

- Milvus
- Qdrant
- Pinecone
- Weaviate
- And more

## Metrics

- Queries per second (QPS)
- Latency (p50, p95, p99)
- Recall accuracy
- Cost per query
- Indexing time
- Memory usage

## Use Cases

- Evaluating vector databases for POC
- Production capacity planning
- Cost optimization
- Performance validation

## Overview

Firecrawl is an API that scrapes, crawls, and extracts structured data from any website, powering AI agents and apps with real-time context from the web. Firecrawl is tailored for LLM engineers, data scientists, AI researchers, and developers looking to harness web data for training machine learning models, market research, content aggregation, and more.

## Key Features

Firecrawl is designed with reliability and AI-ready data in mind, focusing on delivering data reliably and in a LLM-ready format - so you can spend less tokens and build better AI applications.

The platform offers several capabilities:
- Covers 96% of the web, including JS-heavy pages.
- Delivers results in less than 1 second, fast for real-time agents and dynamic apps.
- Unlike traditional web scrapers, Firecrawl is equipped to handle dynamic content rendered with JavaScript and ensures comprehensive data collection from all accessible subpages.

## AI Integration

The Firecrawl Skill is an easy way for AI agents such as Claude Code, Antigravity and OpenCode to use Firecrawl through the CLI. The MCP Server connects Firecrawl directly to Claude, Cursor, Windsurf, VS Code, and other AI tools.

Firecrawl /agent is a magic API that searches, navigates, and gathers data from even the most complex websites. Describe what data you want and agent handles the rest.

## Overview

Vespa is an open-source big data serving engine for vector search and ML.

## Hybrid Capabilities

- Vector ANN + lexical (BM25-like) + structured search.
- Graph/tensor ops for complex relations.
- Built-in phase ranking with ML rerankers.

## Features

- ANN vector search
- Tensor computations
- Hybrid text/structured search
- Real-time ML ranking

## Use Cases

- Multimodal RAG with hybrid retrieval.
- E-commerce recommendations.

## Comparisons vs Pure Vector DBs (e.g., Qdrant)

| Feature | Vespa | Pure Vector DB |
|---------|-------|----------------|
| Hybrid Search | Full lexical+vector+rerank | Vector primary |
| ML Serving | Native | External |
| Real-time | Updates+ranks | Indexing |

## Overview

UNIFY introduces a unified index structure that supports efficient range-filtered approximate nearest neighbor search.

## Key Contributions

- Unified index combining vector and range filtering
- Efficient retrieval under range constraints
- Single structure handles multiple query types
- Published in VLDB 2025

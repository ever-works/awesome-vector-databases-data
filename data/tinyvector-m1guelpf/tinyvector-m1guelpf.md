## Overview
tinyvector provides efficient on-device vector search.

## Features
- In-memory HNSW-like indexing
- Scales to 100M+ vectors
- Axum HTTP server

## Use Cases
- Edge document chat
- IoT website search
- Real-time CV

## Comparisons
- Tiny/simple vs Qdrant edge
- Better accuracy on medium datasets

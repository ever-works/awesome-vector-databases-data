## Overview
Sonic enables low-latency edge vector/hybrid search.

## Features
- HNSW vector + full-text
- In-memory sub-ms ops
- HTTP/JSON API
- Multi-threaded

## Use Cases
- IoT real-time AI
- Edge semantic search
- CV inference

## Comparisons
- Lightweight on-device vs cloud Qdrant
- Faster ingest than heavier DBs

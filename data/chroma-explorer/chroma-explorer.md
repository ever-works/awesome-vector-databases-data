## Overview

Chroma Explorer is a native macOS desktop application that provides a graphical user interface for managing ChromaDB vector databases locally or remotely.

## Features

- Native macOS application with intuitive interface
- Browse and manage ChromaDB collections
- View and edit documents and metadata
- Inspect vector embeddings
- Query testing and debugging
- Local and remote database support
- Real-time data visualization
- Export and import capabilities

## Key Capabilities

- **Collection Management**: Create, delete, and modify collections with ease
- **Document Browser**: View documents with their embeddings and metadata
- **Query Interface**: Test similarity searches and view results
- **Metadata Editor**: Edit document metadata directly in the UI
- **Connection Manager**: Connect to multiple ChromaDB instances

## System Requirements

- macOS 11.0 or later
- ChromaDB instance (local or remote)

## Use Cases

- Development and debugging of ChromaDB applications
- Data exploration and analysis
- Database administration tasks
- Educational purposes for learning vector databases

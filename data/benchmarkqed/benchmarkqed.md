## Overview

BenchmarkQED is an open benchmarking framework for Retrieval-Augmented Generation (RAG) systems designed to push the community toward fairer, comparably measured retrieval evaluation methods.

## Features

- Standardized RAG evaluation protocols
- Focus on retrieval component fairness
- Comprehensive metrics including QPS and latency
- Open-source and extensible

## Use Cases

- RAG system selection and optimization
- Vector DB integration testing
- Production performance validation

Free and open-source.

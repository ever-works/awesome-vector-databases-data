## Overview

MP-RW-LSH provides an efficient multi-probe LSH approach for A1-norm (Manhattan distance) nearest neighbor search.

## Key Contributions

- Multi-probe strategy for LSH under A1-norm
- Efficient hash function design
- Improved retrieval accuracy for L1 distance
- Published in VLDB 2021

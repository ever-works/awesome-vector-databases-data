## Overview

Starling presents an I/O-efficient disk-resident graph index framework designed for high-dimensional vector similarity search.

## Key Contributions

- Disk-resident graph index with I/O optimization
- Segment-based data organization for efficient loading
- Handles billion-scale datasets on single nodes
- Published in SIGMOD 2024
